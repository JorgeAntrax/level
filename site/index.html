Primeros pasos

Si estas utilizando el pluginEmmeten tu editor de código, probablemente las siguientes meta-etiquetas se generen automatícamente, con excepción de la etiquetaviewport.

1.- Actualmente más del 70%de los sitios web existentes en internet son construidos bajo los últimos lineamientos que incorpora laW3C, por este motivo debes incluir en tu documentoindex.htmlla meta etiqueta doctype de html5.

<!DOCTYPEhtml>

2.- Es recomendable incluir la etiqueta para la codificación de caracteres, con el fín evitar símbolos extraños en tus proyectos.

<metacharset"utf-8">

3.- En los ultimos años la necesidad de adaptar tus proyectos a diferentes dispositivos es practicamenteobligatoríoincluir en tu plantillahtml5la meta etiqueta viewport.

<metaname="viewport"content="width=device-width, user-scalable=no, initial-scale=1.0, maximum-scale=1.0, minimum-scale=1.0">

Función Grid

Permite controlar el número de columnas que necesites construir.

grid($columns,$total-columns);
  • $columns (numero de columnas que abarca un elemento).
  • $total-columns (total de columnas en el grid).

Permite convertir la unidad depixelesen la unidad relativarem.

rem($size,$unity);
  • $size (medida a convertir en pixeles).
  • $unity (tamaño de fuente).
  • Si$unity no se especifica por defecto su valor es determinado por la variable $text-default del archivo _variables.scss

Breakpoints soportados
  • mobile= screen < 480px
  • touch= screen < 768px
  • only-touch= screen > 480px < 767px
  • tablet= screen > 768px
  • only-tablet= screen > 769px < 1023px
  • desktop= screen > 1024px
  • only-desktop= screen > 1024px < 1299px
  • widescreen= screen > 1300px
// Revisa del lado izquierdo los breakpoints soportados.@include since ($breakpoint){
  display: flex;
}
@include until ($breakpoint){
  display: flex;
}
@include only ($breakpoint){
  display: flex;
}
Importante
  • Los breakpointsonly-touch, only-tablet, only-desktopson exclusivos del mixin only().
  • Cualquiera de los 3 mixins soportan valores, que no esten dentro de los breakpoints soportados.
  • Los breakpoints pueden ser personalizados modificando el archivo
    • _variables.scss
